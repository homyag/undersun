# Настройка автоматического перевода контента

Система поддерживает автоматический перевод контента блога и недвижимости с русского языка на английский и тайский через API Google Translate или DeepL.

## Настройка API ключей

### Вариант 1: Google Translate API (рекомендуется)

1. Создайте проект в [Google Cloud Console](https://console.cloud.google.com/)
2. Включите Google Translate API
3. Создайте API ключ или Service Account
4. Добавьте ключ в файл `.env`:

```bash
# Google Translate API
GOOGLE_TRANSLATE_API_KEY=your_google_api_key_here
TRANSLATION_SERVICE=google
```

### Вариант 2: DeepL API

1. Зарегистрируйтесь на [DeepL API](https://www.deepl.com/api)
2. Получите API ключ
3. Добавьте ключ в файл `.env`:

```bash
# DeepL API
DEEPL_API_KEY=your_deepl_api_key_here
TRANSLATION_SERVICE=deepl
```

### Дополнительные настройки

```bash
# Настройки перевода (опционально)
TRANSLATION_SOURCE_LANGUAGE=ru
TRANSLATION_TARGET_LANGUAGES=en,th
TRANSLATION_CHUNK_SIZE=5000
```

## Установка зависимостей

```bash
pip install google-cloud-translate==3.15.3 deepl==1.17.0
```

## Использование в админке

### Для блога:

1. Перейдите в админку → Блог → Статьи блога
2. Выберите статьи для перевода
3. В меню "Действие" выберите:
   - **🌐 Перевести на EN и TH (только пустые поля)** - переводит только пустые поля
   - **🔄 Перевести заново (перезаписать все переводы)** - принудительно перезаписывает все переводы

### Для недвижимости:

1. Перейдите в админку → Недвижимость → Недвижимость
2. Выберите объекты для перевода
3. Используйте те же действия что и для блога

### Отдельный перевод:

В форме редактирования статьи или объекта недвижимости доступна кнопка **"Перевести"** для перевода отдельного элемента.

## Переводимые поля

### Блог:
- Заголовок (title)
- Краткое описание (excerpt) 
- Контент (content) - с сохранением HTML разметки
- SEO заголовок (meta_title)
- SEO описание (meta_description)
- SEO ключевые слова (meta_keywords)
- Alt текст изображения (featured_image_alt)

### Недвижимость:
- Название (title)
- Описание (description) - с сохранением HTML разметки
- Краткое описание (short_description)
- Адрес (address)

### Дополнительные модели:
- **Типы недвижимости**: отображаемое название
- **Застройщики**: название и описание  
- **Характеристики**: название

## Логирование

Все операции перевода логируются. Для просмотра логов включите DEBUG и проверьте логи Django.

## Ограничения API

### Google Translate API:
- Платный сервис после бесплатного лимита
- Лимит: 500,000 символов/месяц бесплатно
- Высокое качество перевода

### DeepL API:
- Лимит Free план: 500,000 символов/месяц
- Более высокое качество перевода, особенно для европейских языков
- Ограниченная поддержка тайского языка

## Безопасность

- Храните API ключи в переменных окружения
- Не коммитьте ключи в репозиторий
- Используйте разные ключи для разработки и продакшена

## Устранение неполадок

1. **Ошибка "API перевода не настроен"**:
   - Проверьте наличие API ключа в `.env`
   - Убедитесь что файл `.env` загружается

2. **Ошибки перевода**:
   - Проверьте интернет-соединение
   - Убедитесь что API ключ валидный
   - Проверьте квоты API

3. **Неправильный перевод**:
   - Используйте "принудительный перевод" для обновления
   - Проверьте исходный текст на корректность
